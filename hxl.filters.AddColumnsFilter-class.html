Composable filter class to add constant values to every row of a HXL dataset.

This is a streaming filter to add constant values, such as a country code, to every row of data. It supports the hxl.model.Dataset.add_columns method and the hxladd command-line utility.

This example will add a column with the label "Country name", the HXL hashtag "#country", and the value "Malaysia" to every row in the dataset:

   filter = AddColumnsFilter(source, "Country name#country=Malaysia")

See Also: ColumnFilter, RenameFilter

Construct a new AddColumnsFilter.

The source parameter may be either a string or a list of strings (for multiple columns). Each string must have the following format (<header text> is optional):

<header text>#<tag and attributes>=<constant value>

Example:

   Country name#country=Malasia

By default, this filter will add new columns to the end of existing ones, but you can use optional before parameter to add them to the front.

Parameters:
  • source - a HXL data source
  • specs - a string or list of strings containing new column specifications, as described above (or a list of tuples as described in parse_spec)
  • before - true to add new columns before existing ones (default False)
