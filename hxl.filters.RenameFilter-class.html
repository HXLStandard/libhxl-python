Composable filter class to rename columns in a HXL dataset.

This is the class supporting the hxlrename command-line utility.

Because this class is a {@link hxl.model.Dataset}, you can use it as the source to an instance of another filter class to build a dynamic, single-threaded processing pipeline.

Usage:

<pre> source = HXLReader(sys.stdin) filter = RenameFilter(source, rename=[[TagPattern.parse('#foo'), hxl.model.Column.parse('#bar')]]) write_hxl(sys.stdout, filter) </pre>
