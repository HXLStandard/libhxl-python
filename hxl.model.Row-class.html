get(self, tag, index=None, default=None, parsed=False)

source code 

Get a single value for a tag in a row. If no index is provided ("None"), return the first non-empty value.

Parameters:
  • tag - A TagPattern or a string value for a tag.
  • index - The zero-based index if there are multiple values for the tag (default: None)
  • default - The default value if not found (default: None). Never parsed, even if parsed=True
  • parsed - If true, use attributes as hints to try to parse the value (e.g. number, list, date)

dictionary(self)

source code 

Return the row as a Python dict. The keys will be HXL hashtags and attributes, normalised per HXL 1.1. If two or more columns have the same hashtags and attributes, only the first will be included.
