Class TagPattern

source code

object --+
         |
        TagPattern

Pattern for matching a HXL hashtag and attributes #tag matches #tag with any attributes #tag+foo matches #tag with foo among its attributes #tag-foo matches #tag with foo *not* among its attributes #tag+foo-bar matches #tag with foo but not bar
