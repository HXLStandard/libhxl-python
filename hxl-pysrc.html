 1  """Support library for the Humanitarian Exchange Language (HXL), version 1.0. 
 2   
 3  This library provides support for parsing, validating, cleaning, and 
 4  transforming humanitarian datasets that follow the HXL standard. Its 
 5  use will be familiar to developers who have worked with libraries like 
 6  U{JQuery<https://jquery.com>}.  Here's an example:: 
 7   
 8    import hxl 
 9    data = hxl.data('data.xlsx', True).with_rows('org=UNICEF').without_columns('contact').count('country') 
10   
11  This two-line script performs the following actions: 
12   
13    1. Load and parse the spreadsheet C{data.xlsx} (the library can also 
14       load from any URL, and understands how to read Google 
15       spreadsheets or U{CKAN<http://ckan.org>} resources). 
16   
17    2. Filter out all rows where the value "UNICEF" doesn't appear under 
18       the C{#org} (organisation) hashtag. 
19   
20    3. Strip out personally-identifiable information by removing all 
21       columns with the C{#contact} hashtag (e.g. C{#contact+name}, 
22       C{#contact+phone}, C{#contact+email}). 
23   
24    4. Produce a report showing the number of times each unique 
25       C{#country} appears in the resulting sheet (e.g. to count the number 
26       of activities being conducted by UNICEF in each country). 
27   
28  To get started, read the documentation for the L{hxl.data} function 
29  and the L{hxl.model.Dataset} class. The various filters are also 
30  available as command-line scripts, so you could perform the same 
31  actions as above in a shell script like this:: 
32   
33    $ cat data.xlsx | hxlselect -q 'org=UNICEF' | hxlcut -x contact | hxlcount -t country 
34

40  @see: U{https://proxy.hxlstandard.org} for web-based deployment of this library 
41  """
