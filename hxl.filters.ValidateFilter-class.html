Composable filter class to validate a HXL dataset against a schema.

This is the class supporting the hxlvalidate command-line utility.

Because this class is a {@link hxl.model.Dataset}, you can use it as the source to an instance of another filter class to build a dynamic, singled-threaded processing pipeline.

Usage:

<pre> source = HXLReader(sys.stdin) schema = hxl.schema('my-schema.csv') filter = ValidateFilter(source, schema) write_hxl(sys.stdout, filter) </pre>
