Add HXL hashtags to a CSV-like input stream.

Usage:

 input = open('data.csv', 'r')
 specs = [('Cluster', '#sector'), ('Organi', '#org'), ('province', '#adm1+es')]
 tagger = Tagger(input, specs)

The tagger object acts as a hxl.io.AbstractInput source, which you can use with the hxl.data function like this:

 source = hxl.data(Tagger(input, specs)).with_rows('org=unicef').sort()

Construct a new Tagger object.

The input spec is a list of tuples, where the first item is a substring to match (case-/space-/punctuation-insensitive), and the second item is the HXL tag spec to use. Example:

 [('Cluster', '#sector'), ('Organi', '#org'), ('province', '#adm1+es')]
Parameters:
  • input - an input stream of some kind.
  • specs - the input specs, as described above (default: [])
