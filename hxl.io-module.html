Function Details

data(data, allow_local=False, sheet_index=None, timeout=None, verify_ssl=True, selector=None)

source code 

Convenience method for reading a HXL dataset. If passed an existing Dataset, simply returns it.

Parameters:
  • data - a HXL data provider, file object, array, or string (representing a URL or file name).
  • allow_local - if true, allow opening local filenames as well as remote URLs (default: False).
  • sheet_index - if supplied, use the specified 1-based index to choose a sheet from an Excel workbook (default: None)
  • timeout - if supplied, time out an HTTP(S) request after the specified number of seconds with no data received (default: None)
  • verify_ssl - if False, don't verify SSL certificates (e.g. for self-signed certs).
  • selector - selector property for a JSON file (will later also cover tabs, etc.)

make_input(raw_source, allow_local=False, sheet_index=None, timeout=None, verify_ssl=True, selector=None)

source code 

Figure out what kind of input to create.

Can detect a URL or filename, an input stream, or an array. Will also try to detect HTML and Excel before defaulting to CSV. The result is an object that can deliver rows of data for the HXL library to parse.

Parameters:
  • raw_source - the raw data source (e.g. a URL or input stream).
  • allow_local - if True, allow opening local files as well as remote URLs (default: False).
  • sheet_index - if a number, read that sheet from an Excel workbook (default: None).
  • timeout - if supplied, time out an HTTP(S) request after the specified number of seconds with no data received (default: None)
  • verify_ssl - if False, don't try to verify SSL certificates (e.g. for self-signed certs).
  • selector - a property to select the data in a JSON record (may later extend to spreadsheet tabs).
Returns:
an object belonging to a subclass of AbstractInput, returning rows of raw data.

open_url_or_file(url_or_filename, allow_local=False, timeout=None, verify_ssl=True)

source code 

Try opening a local or remote resource. Allows only HTTP(S) and (S)FTP URLs.

Parameters:
  • url_or_filename - the string to try openining.
  • allow_local - if True, OK to open local files; otherwise, only remote URLs allowed (default: False).
  • timeout - if supplied, time out an HTTP(S) request after the specified number of seconds with no data received (default: None)
Returns:
an io stream.
